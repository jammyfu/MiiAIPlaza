# Master Plan

## Phase 0: Platform Reframing

- Establish standard project-governance files
- Add architecture and automation docs
- Create `tools/verify.py` and `tools/next_plan.py`
- Define the implementation plan and first acceptance checkpoints

## Phase 1: First Playable Plaza

- Introduce a plaza runtime beside the current editor flow
- Build a third-person 3D scene with traversal
- Add mock residents and hotspot interactions
- Ship a lightweight HUD and mobile-safe controls

## Phase 2: Agent Presence Integration

- Add provider-agnostic client contracts
- Integrate `OpenClaw` through a dedicated adapter
- Add generic adapter seams for other agent backends
- Project presence changes into the world shell

## Phase 3: Social Systems

- Add plaza board and mailbox flows
- Add agent visit and conversation surfaces
- Introduce simple affinity and revisit loops

## Phase 4: Live Ops And Expansion

- Add realtime sync and world event scheduling
- Explore shared sessions or multiplayer
- Build curation and diagnostics tools for long-running operation
